## English

SugarPaw is a health-tracking app for cat caregivers. It helps you record:

- Blood glucose history and trends
- Meals & calories
- Insulin injections
- Weight & lab results
- Care diaries, abnormal symptoms, and photos
- Exportable PDF/PNG reports for vets
- Caregiver sharing for the same cat

`index.html` in this repo is used as the Support URL for App Store Connect.
